# AI Reference Memo | AI 参考备忘录
## Project Overview | 项目概述
Solve AI interruption and voice overlap caused by VAD misjudgment in real-time voice interaction. Replace pure algorithm optimization with visual breathing light cues to coordinate human-AI dialogue rhythm.
解决实时语音交互中VAD误判导致的AI插话、语音重叠问题，以呼吸灯视觉提示替代单纯算法优化，协调人机对话节奏。

## Core Working States & Light Rules | 核心状态与光效规则
1. Listening
Stable soft light, device receiving user voice.
**聆听状态**：灯光柔和稳定，设备正常收音。

2. Pause Evaluation
Light changes color & brightness gradually.
- Short thinking pause: Slight light change, no response triggered
- Long ending pause: Complete light transition
**停顿评估状态**：灯光亮度、色彩缓慢渐变
- 短时思考停顿：光效小幅变化，不触发应答
- 长时发言结束停顿：完成全套光效过渡

3. Responding
Light flashes or brightens suddenly, AI starts to reply.
**应答状态**：灯光快速闪烁/提亮，AI 开始回复。

## Device Adaptation | 设备适配
- Smart Speaker: Ring light
  智能音箱：环形灯带
- Mobile Phone: Screen edge / status bar glow
  手机：屏幕边缘、状态栏光效
- AR Glasses: Weak spot light at view edge
  AR 眼镜：视野边缘微光

Note: Use built-in light modules, no extra hardware required.
说明：复用设备原有灯光组件，无需额外加装硬件。

## Core Advantages | 核心特点
Low deployment cost, wide compatibility, strong scalability; active rhythm guidance via visual feedback.
部署成本低、兼容性强、拓展性高；依靠视觉反馈主动协调对话节奏。

   调校参数，适配不同人群说话习惯
2. Combine with emotion recognition for linked light effects
   结合情绪识别，实现光效联动

   拓展适配多人对话场景
